# web-ssh
web ssh terminal by Spring Boot

Java Spring 实现的浏览器端直接SSH连接目标服务器并支持交互操作。

Web SSH 终端的基本原理：
![](preview/WebSSH.png)

用到的相关依赖和框架：
* SpringBoot, Spring MVC
* WebSocket
* Jsch来实现SSH连接和交互
* 前端主要使用xterm.js：https://github.com/xtermjs/xterm.js
* 参考Python实现：https://github.com/huashengdun/webssh
